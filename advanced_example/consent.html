Advanced BOF Project Example

This example demonstrates many of the capabilities of Bride of Frankensystem. This includes:
  • Showing basic and more advanced questionnaires.
  • A questionniare that includes calculations to be made when exporting data to score questionnaire responses.
  • A questionnaire with a custom question type.
  • Automatically assigning users to treatment groups (they are shown the questionnaires in a different order in this example).
  • Defining custom database tables with export calculations.
  • Including instructions pages.
  • Embedding custom pages that show custom html and add data to the database.
  • Custom admin pages.
This is typically where your consent text will show up.
